# Landing Page
Landing page desenvolvida para apresentação pessoal e prática de Front-End, com ênfase em organização de código, boas práticas e suporte para dois idiomas.

# Funcionalidades
- Troca de idioma **PT/EN** em tempo real, sem descarregar a página;
- O idioma escolhido prevalece mesmo após recarregar a página devido **Local Storage**;
- HTML semântico, limpo e acessível;
- Layout responsivo;
- Código comentado e organizado.

# Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

# Como Rodar o Projeto
Não é necessário instalar dependências. É preciso apenas abrir o arquivo 'index.html' em qualquer navegador.
