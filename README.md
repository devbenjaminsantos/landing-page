# Landing Page
Landing page developed for self-introduction and front-end practice, with enphasis on clean code, good practicies and suport for both English and Portuguses languages.

# Features
- **PT/EN** language switch in real time (no page reload);
- Language preference persisted using **LocalStorage**;
- Semantic and acessible HTML;
Responsive layout;
Organized and comented code.

# Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)

# How to Run
No installation required. Just open 'index.html' file in any modern browser.
